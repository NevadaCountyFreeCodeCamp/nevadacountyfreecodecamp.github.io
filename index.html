About us

Nevada County Free Code Camp Meetup is a casual gathering for anyone who wants to learn how to code, get help on Free Code Camp lessons, or work on open source and other passion projects. Meet other local developers, IT professionals and expand your network. This event is for anyone who already codes or is curious about getting started coding! Each week we will try to feature “Lightning Talks”, short talks by local developers and IT professionals on current coding tips and practices.
